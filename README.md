🌍 AyuGo - Travel and Adventure

AyuGo is a responsive travel and adventure website developed as a university group project to promote tourism and provide travelers with useful information about destinations, travel tips, and user experiences.

📌 Project Overview

This website allows users to explore various travel destinations, browse travel tips, view photo galleries, and interact with an advanced review management system.

The project was developed using front-end web technologies with an emphasis on responsive design, interactive user interfaces, and client-side data persistence.

⭐ Review Management System (My Contribution)

The Review Page was primarily developed and enhanced by Kalindu Vidmal.

Implemented functionalities include:

* Add new reviews
* View all submitted reviews
* Edit existing reviews
* Delete individual reviews
* Highlight reviewer names
* Display only the most recent three reviews by default
* Dynamic review statistics
* Form validation

This module demonstrates CRUD operations, DOM manipulation, event delegation, and client-side storage techniques.
